New Technologies Learned Along the Way

All good projects help us learn new things throughout the process of building out their functionality and design. Each new feature provides another opportunity to discover a terrific new tool. These are the technologies I discovered and implemented while developing these projects.

  • Ruby on Rails & React tech stack

    React in the frontend using components to handle all UI interaction and Ruby on Rails, a simple backend framework to handle all database management with Active Record.

  • Tailwind/Flowbite

    Popular CSS framework that allows you to define all your customized UI styling directly in the HTML and makes it convenient to incorporate items from Flowbite, a library providing a large set of prestyled design components.

  • Google APIs

    Well developed tools to provide extensive Map features, effective Autocomplete for a wide variety of searching, and rich data about Places around the world.

  • Cloudinary

    Robust 3rd party media asset hosting site to handle all visual needs, from simple user account images to captivating video with AI capabilities.

  • SerpApi

    Powerful and data rich webscraping api that quickly collects and delivers massive amounts of relevant info, primarily making use of the vast results provided by Google.

  • EmailJS

    Easy to set up Email service that can be managed completely on the frontend, no server-side needed. An email template can be made and implemented in a very short time.
